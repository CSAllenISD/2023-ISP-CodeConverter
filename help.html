As of now, the only languages able to be translated are swift and C++. We declared this as our minimal viable product. In the future we will enable other programs to be able to translate

Our mission in creating this application is valuing optimization while respecting simplicity. Some programs can be optimized easier. Other codes are easier to understand thus program. Our purpose is to maximize both these parameters

There are 2 ways to input your Swift program. Either uploading a swift file or copying and pasting our program in the designated box

Once your program has been translated, there will be a save file button on the C++ desginated box. That allows you to save the translated program as a C++ file.

As of now the application is free, as we are still testing its validity in translation. In the future, there is a chance our team will employ a payment plan to translate code.

As the development team, we predict at least a 95% translation validity in the application? Though tests are still being made.

As of now we only have Swift and C++. But soon, JavaScript and Python will be added soon, as those translation programs are being worked upon as we speak. Although, the JavaScript and Python are not part of our minimal viable product that will be released?
